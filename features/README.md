# 功能開發清單

每個功能獨立一份 `.md` 檔案，記錄背景、調查結論、決策與實作計畫。

## 狀態說明

| 標籤 | 意義 |
|---|---|
| 🔍 調查中 | 可行性尚在評估 |
| 📋 待開發 | 已確認方向，等待排程 |
| 🚧 開發中 | 正在實作 |
| ✅ 已完成 | 已上線 |
| ❌ 放棄 | 評估後不做，附原因 |
