## 計算方法

加權指數（TAIEX）採市值加權：

```
指數 = 總市值 ÷ 除數
```

某支股票漲 1 元對指數的影響：

```
影響點數 = 佔比 × 大盤指數 ÷ 股價
         = 流通股數 ÷ 除數
```

關鍵：影響力取決於**流通股數**，與股價成反比。高股價的大型股，每漲 1 元的點數影響反而不一定最大。

## 重要觀察

- **台積電漲 1 元** → 大盤 +5.71 點；但台積電若漲 **100 元** → 大盤約 +571 點
- **台塑化、金融股**（低股價、高股本）每漲 1 元對大盤的邊際衝擊反而更大
- **聯發科、台達電**雖在前十大，但因股價逾 2,000 元，每漲 1 元幾乎感受不到
- 正確觀察市場走勢應看**漲跌幅**（%），而非絕對漲跌元

## 注意事項

- 佔比資料每月底更新（來源：HiStock 加權指數成分股）
- 股價每日變動，實際影響點數也隨之變動
- 上述計算為理論值，實際交易中會有小誤差（浮動除數、指數暫停計算等）
